# Advanced Linux Administration

## Course

IT 370 – Advanced Linux

## Overview

This course strengthened my understanding of Linux operating systems and server administration.

## Skills Learned

- User Management
- File Permissions
- Bash Scripting
- Cron Jobs
- System Monitoring
- Security Hardening

## Challenges

Learning advanced Linux commands and troubleshooting system issues required patience and consistent practice.

## What I Learned

The course taught me that preparation is one of the most important principles in system administration.

Alma 48 reminds me of Captain Moroni strengthening weak places before the enemy attacked. Likewise, Linux administrators should secure systems before vulnerabilities are exploited.
